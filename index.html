    How to update (no HTML edits)

    DonDominio / FTP friendly
    1. Update prices / models: edit /data/catalog.json (with Notepad). Save and upload it.
    2. Update the top banner message: edit /data/message.txt. Save and upload it.
    3. Add images: upload JPG/PNG to /assets/models/ and set that path in JSON (example: assets/models/sl4802-skeleton.jpg).

    If something does not load, check: file names, uppercase/lowercase, and that the server allows serving .json and .txt.
